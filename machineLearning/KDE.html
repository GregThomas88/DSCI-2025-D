Cleaning the source data to remove null values and turning it into nice numbers for comparison

In[8]:
nullVal = r"\N"
routes = routes[routes['seconds'] != nullVal]

routes['seconds'] = routes['seconds'].astype(int)

routes['hours'] = round(routes['seconds']/3600, 0)

When doing intial data analysis, histograms can provide great value for a viewer to understand the distribution of data

In[9]:
plt.hist(data=routes, x="hours")
plt.show()

Compared to the histogram, the KDE plot below shows a similar distribution, but has a more accurate shape of the data, since it is using a gaussian distribution kernel function on every data point and then taking the sum. The one thing to watch out for is the small outliers in the data.

In[10]:
sns.kdeplot(data=routes, x='hours', bw_adjust=0.3)
